Algorytm A*

Jest to jeden z najbardziej popularnych algorytmów odnajdywania ścieżki. Algorytm ten jest powszechnie używany między innymi w grach komputerowych. Zadaniem tego algorytmu jest jak najszybsze znalezienie drogi pomiędzy punktem startowym a docelowym. Posiada on trzy parametry:
g: Koszt przemieszczenia się z punktu startowego do obecnego, przedstawiony jako suma dotychczasowych punktów odwiedzonych przez algorytm.
h: Znane również jako wartość heurystyczna, jest to szacowany koszt przejścia z obecnej komórki do komórki końcowej. Koszt rzeczywisty nie zostaje wyliczony do momentu dotarcia do komórki końcowej.
f: To suma parametrów g oraz h. Podczas działania algorytmu podejmuje on decyzję biorąc pod uwagę wartość parametru f. Wybiera on komórkę, w której jego wartość jest najniższa i do niej się przemieszcza. Proces ten powtarza się, dopóki algorytm nie osiągnie swojego celu.

Przykładowa implenetacja


def bubbleSort(arr):
n = len(arr)

# Traverse through all array elements
for i in range(n-1):
# range(n) also work but outer loop will repeat one time more than needed.

    # Last i elements are already in place
    for j in range(0, n-i-1):

        # traverse the array from 0 to n-i-1
        # Swap if the element found is greater
        # than the next element
        if arr[j] > arr[j + 1] :
            arr[j], arr[j + 1] = arr[j + 1], arr[j]

# Driver code to test above
arr = [64, 34, 25, 12, 22, 11, 90]

bubbleSort(arr)
